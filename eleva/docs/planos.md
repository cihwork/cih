**Data**: 2026-04-07
**Objetivo**: Definir estratégia de precificação SaaS + Consultoria para o Sistema Operacional de RH

## 🎯 Posicionamento de Mercado

**Não somos**: HRIS tradicional
**Somos**: Sistema operacional de liderança com PDI vivo + IA assistida

**Modelo de Negócio**: SaaS + Consultoria (híbrido)

**Diferencial**:
- PDI contínuo (não anual)
- DISC + EC (calibração por maturidade)
- Perguntas comparativas inteligentes (RAG)
- Validação cruzada (anti-viés)
- Recomendações de IA para liderança
