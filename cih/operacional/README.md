# CIH - Operacional

Estrutura operacional do programa de Desenvolvimento Humano e Organizacional.

## Sobre o CIH

O CIH combina **expertise humana** com **inteligência artificial** para entregar um ciclo completo de desenvolvimento de colaboradores - do assessment comportamental ao acompanhamento contínuo.
